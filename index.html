pysynscan¶

A pure python3 implementation of skywatcher synscan motor protocol. Tested on SkyWatcher AZ-GTI mount and open-synscan project

Installation¶

From sources:

git clone https://github.com/nachoplus/pysynscan.git
python3 setup.py install

Use¶

Several code examples are in examples directory

Also some CLI (command lines interface) are provided.

See full documentation on [https://nachoplus.github.io/pysynscan]

Command line utilities¶

synscanGoto¶

Do a GOTO to a target azimuth/altitude

synscanGoto [OPTIONS] AZIMUTH ALTITUDE

Options

--host <host>¶

Synscan mount IP address

--port <port>¶

Synscan mount port

--wait <wait>¶

Wait until finished (default False)

Arguments

AZIMUTH¶

Required argument

ALTITUDE¶

Required argument

synscanTrack¶

Move at desired speed (degrees per second)

synscanTrack [OPTIONS] AZIMUTH_SPEED ALTITUDE_SPEED

Options

--host <host>¶

Synscan mount IP address

--port <port>¶

Synscan mount port

Arguments

AZIMUTH_SPEED¶

Required argument

ALTITUDE_SPEED¶

Required argument

synscanStop¶

Stop Motors

synscanStop [OPTIONS]

Options

--host <host>¶

Synscan mount IP address

--port <port>¶

Synscan mount port

--wait <wait>¶

synscanShow¶

--port <port>¶
